# sakuhin
Excelデータを読み込み、集計・整形・グラフ生成を自動化するPythonスクリプトです。

## Excel 自動化ツール
Excelデータを読み込み、集計・整形・グラフ生成を自動化するPythonツールです。

### 使用ライブラリ
- pandas
- matplotlib
- openpyxl

### 使い方
1. data.xlsx を用意する
2. main.py を実行する
3. summary.xlsx と graph.png が生成されます
